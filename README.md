# Show/Hide Text - jQuery plugin

Simple jQuery plugin show/hide text  with some settings.

## Demo

[demo link](https://rublikfun.github.io/show-hide-text/)

## Settings

```
{
    charQty     : 250,
    ellipseText : "...",
    moreText    : "Read more",
    lessText    : "Hide"
}
```
